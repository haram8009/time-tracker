# LongPressDrag: 롱프레스로 드래그 선택 진입

그리드에서 즉시 드래그하면 세로 스크롤과 셀 선택이 충돌한다. 이를 해소하기 위해 롱프레스(햅틱+시각 피드백)를 드래그 선택 진입 트리거로 채택했다. 롱프레스 없이 손가락을 움직이면 일반 스크롤로 처리된다.

## Considered Options

- **즉시 드래그(기존):** 수직 이동 8px 임계값으로 선택 시작. 빠르지만 스크롤 의도와 충돌이 잦다.
- **별도 모드 토글:** 상단 버튼으로 "편집 모드" 전환. 충돌은 없지만 진입 마찰이 크다.
- **롱프레스 진입(채택):** 자연스러운 "잡고 끌기" 은유. 빈 셀 탭은 단일 셀 선택으로 유지해 빠른 입력도 가능.
